# Phase B Showcase World

This is the curated mining showcase scenario for the Phase B world/render foundation.
It is designed to make the stronger world form, semantic selection, and scene framing obvious at a glance.

## Launch

```bash
python3 -m autonomous_ops_sim.cli live --scenario scenarios/phase_b_showcase/01_mine_phase_b_showcase.json
```

## What It Shows

- a strong operational road spine
- a visible loading area and unloading / crusher area
- a staging / holding pocket
- a clear site boundary and terrain contrast
- a larger scene frame that makes fit / focus / minimap feel useful
- semantic selection targets for vehicles, roads, intersections, and world-form areas

## Intended Limits

- this is a presentation-quality Phase B world, not Phase C traffic realism
- the scenario keeps the frontend on derived world / render surfaces
- the existing flagship mining scenario remains unchanged
